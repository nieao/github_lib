# GitHub Actions with Claude

This repository is configured with Anthropic's official Claude GitHub Actions:

## Features

### 1. Claude Code Action
- Auto PR code review
- Respond to `@claude` mentions in comments
- OWASP security analysis

### 2. Security Review
- AI-powered security vulnerability analysis
- Automatic PR comments with findings
- Only analyzes changed files

## Setup

1. Add `ANTHROPIC_API_KEY` to repository secrets
2. Create a PR to trigger the actions

## Usage

- Mention `@claude` in any PR or issue comment
- Open a PR to trigger automatic security review
